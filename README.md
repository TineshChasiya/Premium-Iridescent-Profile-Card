# Premium Iridescent Profile Card

A modern **Premium Iridescent Profile Card** built entirely with **HTML and CSS**. This project demonstrates advanced glassmorphism techniques using frosted glass effects, iridescent gradients, dynamic lighting, and smooth hover interactions to create a clean and professional user interface.

## ✨ Features

- Premium Glassmorphism Design
- Iridescent Gradient Overlay
- Realistic Frosted Glass Effect
- Animated Ambient Background Orbs
- Smooth Hover Animations
- CSS Grid Layout
- CSS Variables for Easy Customization
- Responsive and Clean Structure
- Beginner-Friendly Code

1. Clone or download this repository.
2. Open the project folder.
3. Open `index.html` in your browser or use Live Server.
4. Customize the content, colors, and profile image as needed.
